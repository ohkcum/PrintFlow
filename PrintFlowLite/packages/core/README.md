# PrintFlowLite Core

Core business logic and persistence layer.

## Build

```bash
mvn clean install -DskipTests
```

## Dependencies

- printflowlite-common
- JPA / Hibernate
- Apache Derby (embedded)
- PostgreSQL (production)
