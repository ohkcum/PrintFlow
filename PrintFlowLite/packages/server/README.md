# PrintFlowLite Server

The main web server application built with Apache Wicket and Jetty.

## Build

```bash
mvn clean package -DskipTests
```

## Run

The WAR file can be deployed to any Servlet 4.0 container (Jetty, Tomcat, etc.)
or run directly using the embedded Jetty launcher.
