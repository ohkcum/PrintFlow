### Dependencies

#### Build

Make sure to build [https://gitlab.com/printflowlite/xmlrpcpp]() in the sibling directory `../xmlrpcpp` first.

The CUPS development package is needed to build the binary. Install with this command:

    $ sudo apt-get install libcups2-dev
